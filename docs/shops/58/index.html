優惠內容

持員工卡至桃園福容霸飯店福粵樓用餐享95折優惠。 注意事項:
1.優惠限10人(含)以下用餐，須另加原價一成服務費，且不可與其他優惠合併使用。
2.加入福容會員再享分級折扣新福級95折/銀福級9折/金福級88折。
3.宴會廳、喜宴、包廂、酒水飲料、商業午餐、外燴、外帶、禮盒、客房餐飲服務、餐券及特殊主題、促銷活動等均不適用。
4.一般平假日皆適用，惟於「農曆春節期間（小年夜至大年初五）、中/西情人節、母親節當日及前一日、端午節、父親節、中秋節、聖誕夜、聖誕節、跨年」等日期不適用（特殊節日定義依飯店規定為準）。
5.持卡消費請事先預約，如遇各分店舉辦特殊活動（如舉辦美食活動或包場活動等），無法提供此優惠時，恕不另行通知。
6.優惠恕無法與其他專案合併使用相關使用規章需依現場公告為主。
7.福容大飯店保留調整或終止活動之權益，相關使用規範需依現場公告為主
