優惠內容

1.每位員工符合資格者給予$5,000隱適美隱形牙套優惠折扣碼(僅限領取1次)。
2.本專案如優惠折扣碼餘額已用盡，得提前終止提供優惠折扣碼。
3.折扣碼之使用條件如下： (1)優惠碼僅供本府員工在診所接受隱適美矯正系統療程折抵牙套工本費使用，優惠碼需由診所代為兌換牙套工本費折抵。 (2)診所製作牙套訂單時需輪入優惠碼，且需在個別員工取得優惠折扣碼之日起3個月内交付牙套製作訂單，交付牙套製作之時間依隱適美台灣之紀錄為依據，逾期或未交付製作隱適美牙套，優惠碼將視為無效代碼。 (3)每人僅可使用1次優惠折扣碼，本優惠折扣得使用之隱適美產品為：成人或孩童之隱形矯正，依據牙醫師專業判斷給予相應對之牙套數量。
4.員工使用優惠碼前應自行向所選擇的診所進行預約，並在療程開始前與診所確認優惠碼適用情形(如遇個別診所不接受此優惠折扣碼，員工可洽詢其他診所使用此優惠折扣碼)。

資料來源：桃園市政府員工卡特約商店名單（2024/12/17 更新）
